# Pak Navy Website Clone

This repository contains a clone of the Pakistan Navy website made using HTML, CSS, and a little JavaScript.
The project is currently **not responsive**, but I will make it responsive soon.

## Features

* Static layout inspired by the Pakistan Navy website
* HTML, CSS, and minimal JavaScript
* Clean structure and organized files

## Upcoming Improvements

* Full responsiveness for mobile, tablet, and desktop
* Improved navigation bar
* Better spacing and layout adjustments
* Smooth animations and transitions

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Project Structure

```
pak-navy-clone/
│── index.html
│── /css
│── /js
└── README.md
```

## How to Run

1. Clone the repository:

   ```
   git clone https://github.com/SultanAli41/Pak-Navy-Clone
   ```
2. Open **index.html** in any browser.
